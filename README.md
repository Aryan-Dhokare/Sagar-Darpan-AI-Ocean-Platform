#  Sagar Darpan – AI Ocean Platform

An AI-powered unified platform for ocean monitoring, fisheries intelligence, and biodiversity analytics. This platform integrates AI, data visualization, and user-friendly interfaces to support fishermen, researchers, and administrators.

---

##  Features

*  AI Fish Identification (Matsya Chatbot)
*  Ocean Health Dashboard
*  Fisherman Advisory System
*  Research Analytics Portal
*  Admin Control Panel

##  How to Run Locally

```bash
# Clone the repository
git clone https://github.com/Aryan-Dhokare/Sagar-Darpan-AI-Ocean-Platform.git

# Navigate to frontend
cd frontend

# Install dependencies
npm install

# Run development server
npm run dev
```

Then open in browser:

http://localhost:3000
